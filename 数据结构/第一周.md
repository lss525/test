# 数据结构    
  
## 概念
  - 数据结构是计算机存储、组织数据的方式，它研究的是：

    - 数据的逻辑结构：数据元素的关系

    - 数据的存储结构：数据与计算机内存的关系
  
## 分类：  
1.逻辑结构  
  -  线性结构  
     - 线性表（数组、链表）    
     - 栈
  队列
  字符串  

  - 非线性结构:  

    - 树形结构
    - 图状结构  

2.存储结构 :
  - 顺序存储
  - 链式存储
  - 索引存储
  - 散列存储    
  
## 逻辑结构：  
### 数组：  
特点：

  - 随机访问：通过下标直接访问，时间复杂度O(1)

  - 内存连续：所有元素在内存中连续存放

  - 固定大小：静态数组大小在编译时确定

操作时间复杂度：

  - 访问：O(1)

  - 插入：平均O(n)（需要移动元素）

  - 删除：平均O(n)

  - 查找：无序时O(n)，有序时O(log n)（二分查找）  
### 链表：  
 概念：  
   - 链表是一种线性数据结构，由一系列节点（Node）通过指针（或引用）链接而成。每个节点包含两部分：

  - 数据域：存储实际数据

  - 指针域：存储下一个节点的地址

 关键思想：  
  - 与数组需要连续内存空间不同，链表采用分散存储策略：

    - 每个节点可以存储在内存的任意位置

    - 通过指针将这些离散的节点串联起来
